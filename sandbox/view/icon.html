Règles de construction des svg pour les fonts :
  • La taille de base est 1024x1024
  • Le contenu est bord à bord en hauteur ou en largeur
La nomenclature :
  • -full : icone pleine
  • -square : le carré plein avec les bords arrondis
  • -sign : le cercle plein
  • -circle : le cercle
Il est possible de jouer sur la tailler de l'icône avec icon-lg et icon-2x à icon-5x
